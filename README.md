# Random Position Generator

Générateur aléatoire de positions pour couples. Simple, fun, sans prise de tête.

## Utilisation

Ouvrir `index.html` dans un navigateur. C'est tout.

- La page génère une position aléatoire au chargement
- Cliquer sur le bouton pour en générer une nouvelle
- Illustrations en stick figures SVG

## Tech

Zero dépendance. Un seul fichier HTML avec CSS et JS embarqués.
